## 템플릿엔진&View 
- 뷰 템플릿(=템플릿 엔진) : 화면을 담당하는 기술로, 하나의 틀로 만들고 여기저기에 변수를 삽입할 수 있게 한다.
    * 종류 : JSP, 타임리프, 머스테치(Mustache)
- MVC 패턴 : Model에 데이터를 담아, View에 보여준다. 이를 Contorller에서 작업
- 디자인 적용 : 1. 레이아웃, 화면에 요소를 배치하는 일을 말한다, 헤더-푸터 레이아웃이 근본
